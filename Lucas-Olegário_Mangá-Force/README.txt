Projeto desenvolvido utilizando *HTML, CSS e JavaScript puro*, com o objetivo de criar um site interativo para exibição de mangás.  
O site possui página inicial, catálogo, detalhes de cada mangá, contato e recursos multimídia.

Funcionalidades:
- Página inicial com "carrossel" e cards de mangás.  
- Página de "catálogo" listando todos os mangás disponíveis.  
- Página de "detalhes* para cada mangá, com sinopse, imagem e áudio.  
- Página de "contato" com formulário e informações adicionais.  
- Navegação dinâmica e intuitiva entre as páginas.  
- "Footer personalizado" com links e redes sociais.  
- Áudios integrados para leitura de sinopses (suporte a bibliotecas de áudio).  
- Header com imagem que muda dependendo do mangá selecionado.  
- Botão de voltar estilizado com animação.

Tecnologias Utilizadas:
- HTML5
- CSS3
- JavaScript

Como Usar:

1. Abra o arquivo "index.html" em um navegador.  
2. Clique em um card para abrir a página de "detalhes":  
   - Veja a sinopse, imagem em destaque e, se disponível, clique no *áudio* para ouvir.  
   - Use o botão "← Voltar" para retornar à página anterior.  
3. No menu de navegação (header), você pode acessar:  
   - "Início" → retorna para a página inicial.  
   - "Catálogo" → exibe a lista completa de mangás.  
   - "Contato" → abre a página de formulário e redes sociais.  
4. Role até o final de qualquer página para ver o "footer" com links rápidos e redes sociais
